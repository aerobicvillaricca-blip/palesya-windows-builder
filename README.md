# Palesya Windows Builder

Repository tecnico pubblico per compilare l'installer Windows di Palesya con i runner GitHub gratuiti.

- Non contiene il codice sorgente di Palesya.
- Legge la repository privata con una deploy key in sola lettura.
- Non contiene dati delle palestre, credenziali cliente o configurazioni hardware.
- Le build partono esclusivamente con avvio manuale del proprietario.
